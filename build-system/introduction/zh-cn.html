WebAssembly (Wasm) 是一种通用的可以运行在 web 上的低级字节码。
Rust, AssemblyScript (类似 Typescript), Emscripten (C/C++) 等多种语言 都可以编译为 Wasm。 Wasm 提供了一种性能可观的紧凑二进制格式，易于移植并与 JavaScript 等宿主语言配合良好。 Wasm 目前正在陆续登陆主流浏览器，也将提供用于运行服务器程序或与系统交互的独立 runtime。

Wasm By Example 是什么？

Wasm By Example 是一个简明的实操教程，通过代码片段与带注解的例程来教学 WebAssembly。 更多关于 Wasm 的信息请参考 WebAssembly Introduction 或浏览下列示例。
